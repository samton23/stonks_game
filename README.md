# 📈 STONKS GAME

**Интерактивная бизнес-симуляция для офлайн игр с Telegram интеграцией**

Полнофункциональная система для проведения увлекательных бизнес-игр, где участники управляют виртуальными предприятиями, зарабатывают прибыль и конкурируют за первое место в реальном времени.

---

## ✨ Возможности

- 🎮 **Telegram Mini App** — удобный игровой интерфейс для участников
- 👨‍💼 **Административная панель** — полный контроль над ходом игры
- 📊 **Live Dashboard** — публичный экран с рейтингом и анимациями
- 💰 **Система предприятий** — разные типы бизнесов с уникальными характеристиками
- 🏭 **Заводы** — улучшения, увеличивающие прибыль
- 🔔 **Уведомления в реальном времени** — Telegram + in-app notifications
- 🎨 **Современный UI** — анимации, градиенты, отзывчивый дизайн
- 🐳 **Docker Compose** — простое развёртывание одной командой

---

## 🎯 Как это работает

1. **Ведущий** регистрирует игроков через админ-панель
2. **Игроки** получают стартовый бюджет и открывают мини-приложение в Telegram
3. **Ведущий** управляет игрой: добавляет предприятия, заводы, переключает циклы
4. **Система** автоматически начисляет прибыль каждый цикл
5. **Дашборд** показывает рейтинг игроков в реальном времени
6. **Победитель** определяется по наибольшему доходу

### Игровая механика

- **Предприятия** генерируют базовую прибыль каждый цикл (или каждые N циклов)
- **Заводы** увеличивают прибыль предприятия на процент
- **Формула дохода:** `прибыль × (1 + кол-во_заводов × процент_завода / 100)`
- **Пример:** база 500, 2 завода по 15% = 500 × (1 + 2 × 15%) = 650 за цикл

## 🚀 Быстрый старт

### Требования

- Docker и Docker Compose
- Telegram бот (токен от [@BotFather](https://t.me/BotFather))
- Домен с HTTPS (обязательно для Telegram Mini Apps)

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone <your-repo-url>
cd stonks_game
```

2. **Настройте переменные окружения:**
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
BOT_TOKEN=your_bot_token_here
WEBAPP_URL=https://yourdomain.com/app
WEB_PORT=3000
```

3. **Запустите систему:**
```bash
docker-compose up -d
```

4. **Проверьте статус:**
```bash
docker-compose ps
```

Все сервисы должны быть в статусе `Up`.

### Доступ к интерфейсам

- **Админ-панель:** `http://your-server:3000/admin`
- **Дашборд:** `http://your-server:3000/dashboard`
- **Telegram бот:** Найдите вашего бота в Telegram и отправьте `/start`

## 🎮 Использование

### Для ведущего игры

1. Откройте админ-панель
2. Добавьте игроков (имя + Telegram ID)
3. Отправьте приглашения
4. Настройте предприятия и стартовый бюджет
5. Запустите игру и переключайте циклы
6. Управляйте игроками: добавляйте предприятия, заводы, изменяйте баланс

### Для игроков

1. Откройте бота в Telegram
2. Нажмите "🎮 Открыть игру"
3. Следите за балансом и доходом
4. Используйте калькулятор для планирования покупок
5. Читайте правила во вкладке "Правила"

### Для зрителей

Откройте дашборд на проекторе — рейтинг обновляется каждые 3 секунды с плавными анимациями.

## 🌐 Production развёртывание

### Настройка HTTPS

Telegram Mini Apps требуют HTTPS. Используйте:

- **Nginx Proxy Manager** — простой GUI для SSL
- **Traefik** — автоматические сертификаты Let's Encrypt
- **Caddy** — автоматический HTTPS
- **Cloudflare Tunnel** — бесплатный туннель без открытых портов

Пример с Nginx:
```nginx
server {
    listen 443 ssl;
    server_name yourdomain.com;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### Безопасность

- Используйте сильные пароли для PostgreSQL
- Не коммитьте `.env` в репозиторий
- Настройте firewall на сервере
- Регулярно обновляйте Docker образы
